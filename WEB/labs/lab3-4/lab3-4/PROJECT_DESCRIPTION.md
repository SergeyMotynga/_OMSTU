# Описание проекта Lab 3-4: Приложение Нобелевских премий

## Обзор проекта

Это веб-приложение на Vue 3 с TypeScript для просмотра информации о Нобелевских премиях и лауреатах. Приложение использует официальный API Нобелевской премии для получения данных и отображает их в табличном виде с возможностями фильтрации, поиска и пагинации.

## Архитектура и технологии

### Основные технологии
- **Vue 3** с Composition API
- **TypeScript** для типизации
- **Vite** как сборщик проекта
- **Vue Router** для навигации
- **Axios** для HTTP-запросов
- **io-ts** и **fp-ts** для валидации и декодирования данных API

## Порядок работы приложения

### 1. Инициализация приложения
1. `main.ts` создает экземпляр Vue приложения
2. Подключает роутер и монтирует приложение в DOM
3. `App.vue` определяет общий layout с навигацией

### 2. Навигация и маршрутизация
- Приложение имеет два основных маршрута:
  - `/laureates` (по умолчанию) - страница лауреатов
  - `/prizes` - страница премий
- Vue Router управляет отображением соответствующих компонентов

### 3. Загрузка данных (на примере страницы лауреатов)

#### 3.1 Инициализация сервисов
```typescript
// src/init/servicesNobel.ts
export const nobelService = new NobelService({
  ajaxService: ajaxNobelService
});
```

```typescript
// src/init/ajaxNobelService.ts
export const ajaxNobelService = new AjaxService('https://api.nobelprize.org/2.1');
```

#### 3.2 Запрос данных
1. `NobelView.vue` вызывает `nobelService.fetchLaureates(params)`
2. `NobelService` делегирует запрос `NobelApiRepository`
3. `NobelApiRepository.getLaureates()` делает GET-запрос к `/laureates`
4. `AjaxService` использует Axios для выполнения HTTP-запроса

#### 3.3 Обработка ответа
1. Полученный JSON декодируется с помощью `decodeOrThrow`
2. `LaureatesResponseCodec` валидирует структуру данных:
   ```typescript
   export const LaureatesResponseCodec = t.type({
     laureates: t.array(LaureateCodec),
   });
   ```
3. `LaureateCodec` определяет структуру лауреата:
   - `knownName.en` - известное имя
   - `fullName.en` - полное имя (опционально)
   - `birth.date` - дата рождения (опционально)
   - `nobelPrizes` - массив ссылок на премии

### 4. Обработка и отображение данных

#### 4.1 Клиентская фильтрация
- Данные фильтруются на клиенте по:
  - Году премии
  - Поиску по имени (в NobelView)
  - Категории (в обоих представлениях)

#### 4.2 Формирование данных для таблицы
- Данные преобразуются в формат, подходящий для `DataTable` компонента
- Каждая строка таблицы содержит отображаемые поля

#### 4.3 Пагинация
- Реализована клиентская пагинация
- Пользователь может выбирать количество элементов на странице (5, 10, 20, 50)
- Отображаются контролы навигации и информация о текущей странице

### 5. Компоненты

#### DataTable.vue
- Универсальный компонент таблицы
- Принимает props: `columns` (массив заголовков) и `rows` (массив объектов данных)
- Автоматически генерирует HTML-таблицу с чередованием цветов строк

#### NobelView.vue и PrizesView.vue
- Основные страницы приложения
- Управляют состоянием фильтров, пагинации и загрузки
- Используют реактивные переменные Vue для управления UI

## Особенности реализации

### Type Safety
- Полная типизация с TypeScript
- Runtime валидация данных API с io-ts
- Строгие типы для всех структур данных

### Архитектурные принципы
- **Separation of Concerns**: Разделение на слои (API, бизнес-логика, представление)
- **Dependency Injection**: Сервисы инициализируются с зависимостями
- **Single Responsibility**: Каждый класс отвечает за одну задачу

### Обработка ошибок
- HTTP-ошибки логируются в консоль
- При ошибках данные сбрасываются в пустые массивы
- Пользователь видит состояние загрузки

### UX/UI
- Адаптивный дизайн с простым CSS
- Индикатор загрузки во время запросов
- Четкая навигация между разделами

## API взаимодействия

Приложение взаимодействует с Nobel Prize API v2.1:
- `GET /laureates` - получение списка лауреатов
- `GET /nobelPrizes` - получение списка премий

Параметры запроса:
- `limit` - количество элементов
- `offset` - смещение для пагинации
- `name` - поиск по имени лауреата
- `nobelPrizeYear` - фильтр по году премии
